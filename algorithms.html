INSERTION SORT

Schritt 1: Wenn es sich um das erste Element handelt, ist es bereits sortiert.
Schritt 2: Wähle das nächste Element
Schritt3: Vergleiche das Element mit jedem Element in dem bereits sortierten Teil
Schritt 4: Füge das Element dort ein, wo es hinpasst.

SELECTION SORT

1. Wähle das kleinste Element aus dem unsortierten Datensatz aus
2. Füge es am Start ein.
3. Suche nun das kleinste Element im Rest des Datensatzes und füge es nach dem letzten Element im sortierten Teil ein
4. usw.

MERGE SORT

1. Teile den Datensatz in 2 Hälften
2. Sortiere die Teilfelder rekursiv genau so
3. Füge die sorteirten Teilfelder zusammen
