INSERTION SORT

Step 1: Select the first unsorted element. The first element is in itself already sorted, so start at the second.
Step 2: Compare the selected element to every element in the already sorted section of the array.
Step 3: Insert the element in its place
Repeat this until the unsorted section is empty.

SELECTION SORT

Step 1: Select the smallest element in the unsorted section of the array (at the start that is of course the entire array)
Step 2: Insert it at the beginning
Step 3: Now select the next smallest element in the remaining array and bring it to the front after the previous one
Repeat as often as necessary

MERGE SORT

This algorithm follows the motto "Divide and Conquer"
Step 1: Divide the array in half and continue doing that until you are left with parts consisting of two elements max.
Step 2: Compare and sort these small parts within themselves
Step 3: Compare and merge the parts to the complete sorted array

Here is an overview of how they compare to each other:

Insertion Selection Merge
in-place ✔ ✔ ❌
stable ✔ ❌ ✔
adaptive ✔ ❌ ❌
Best Case O(n) O(n²) O(n log n)
Average Case O(n²) O(n²) O(n log n)
Worst Case O(n²) O(n²) O(n log n)
